    ${MOCKED_MODULES_SOURCE_DIR}/opend/dialog/fp_hanfun/hanfun_api

    ${MOCKED_MODULES_SOURCE_DIR}/hanfun_lib/hanfun_lib/inc

)

## Add targets
##

## Link targets
##

## Add tests
